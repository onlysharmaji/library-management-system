## ✨ Features

- **📖 Book Catalog** — Add, edit, delete and search books by title, author, or category
- **👥 Member Management** — Register and manage library members
- **🔄 Issue & Return** — Track book lending with due dates
- **💰 Fine Calculation** — Automatic fine of ₹2/day for late returns
- **🔍 Search & Filter** — Real-time search and category filtering
- **📊 Dashboard Stats** — Live counter animation showing totals
- **🔐 Admin Login** — Secure admin panel (demo: admin / admin123)
- **💾 Persistent Data** — Data saved in LocalStorage across sessions
- **📱 Responsive Design** — Works on mobile, tablet, and desktop

## 🚀 How to Run

### Option 1 – Open Directly
Just open `index.html` in any modern web browser. No server needed!

### Option 2 – GitHub Pages (Recommended)
1. Upload this folder to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site is live at `https://yourusername.github.io/repo-name`

### Option 3 – VS Code Live Server
1. Install the "Live Server" extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
